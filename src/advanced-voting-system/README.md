## Setup Project

### Install dependencies
The project uses `yarn`, so you will have to run the following command to install all dependencies:
```shell
yarn install
```

### Run Tests
The projects uses Hardhat.js.
To run tests use the following commands: 
```shell
$ npx hardhat test
```
